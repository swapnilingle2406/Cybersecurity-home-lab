Cybersecurity-home-lab

My first cybersecurity home lab - Kali Linux + DVWA + Wireshark

What I Built:
A Cybersecurity Home Lab with Kali Linux and DVWA to learn how web attacks work and understand Network Traffic using wireshark.

Tools Used
- VirtualBox (runs Kali Linux)
- Kali Linux (attacking machine)
- Docker (runs DVWA)
- DVWA(vulnerable target)
- Wireshark (captured network traffic)

Attack Performed:
- SQL Injection
- Command Injection
- Brute Force
- Reflected Cross Site Scripting

What I Learned
- Setting up virtual machines
- Basic Linux commands
- How Attacks works
- Using Wireshark to capture attacks
